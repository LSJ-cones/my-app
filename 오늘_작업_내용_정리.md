## 📊 Post 엔티티 업데이트 내용

### 새로 추가된 필드들
- `status`: 게시글 상태 (DRAFT, PUBLISHED, ARCHIVED)
- `viewCount`: 조회수
- `likeCount`: 좋아요 수
- `dislikeCount`: 싫어요 수
- `category`: 카테고리 연관관계
- `tags`: 태그 목록 (N:M 관계)
- `files`: 첨부파일 목록

### 새로 추가된 메소드들
- `incrementViewCount()`: 조회수 증가
- `incrementLikeCount()` / `decrementLikeCount()`: 좋아요 수 관리
- `incrementDislikeCount()` / `decrementDislikeCount()`: 싫어요 수 관리
- `addTag()` / `removeTag()`: 태그 관리
